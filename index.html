观云视频云 API

播放器及终端用户请使用 uapi.guancloud.com 这个域名代替 api.guancloud.com，

两个域名的不同之处是 uapi.guancloud.com 提供了 API 缓存，而且不强制用户使用 HTTPS 访问。

视频上传 Legacy ¶

视频上传 API 即将弃用，如果需要使用，请将 API 版本号从 1.2 修改为 1.0。
